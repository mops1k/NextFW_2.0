Возможности

Этот фреймворк разрабатывается на чистом энтузиазме и вероятнее всего он далеко не самое оптивальное решение, но прежде чем уходить предлагаем вам изучить его возможности.

  • Система бандлов, которая позволяет писать множество независимых приложений на одном ядре
  • Консольная утилита для генерации различных бандлов, контроллеров, модулей
  • Система расширений для использования сторонних решений
  • RainTPL по умолчания как шаблонизатор
  • RedBeanPHP 4 по умолчанию в качестве ORM
